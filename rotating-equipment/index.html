Add another rotating equipment checklist

Each checklist is just two files: a data file listing the sections/items, and a thin HTML page that loads it. Copy the pattern from any checklist above:

assets/checklists/<name>-data.js rotating-equipment/<name>.html

In the data file, set a unique storageKey and list your sections/items — the shared engine (assets/app.js) handles checking off items, notes, flags, progress, the summary table, sign-off, printing, and saving automatically. Then add a card for it on the home page and here.
